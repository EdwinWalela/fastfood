# FastFood Restaurant

A frontend application for a fictional restaurant

## Tech Stack
- ReactJS
- React redux (redux toolkit)
- TailwindCSS

## Features
- User authentication
- Dinner reservation
- Online ordering

## Local development

Clone repository

```bash
git clone https://github.com/EdwinWalela/fastfood
``` 

Install dependancies

```bash
cd fastfood && npm i
```

Start development server

```bash
npm start
```
